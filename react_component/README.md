React component

Learning Objectives

By the end of this project, you should be able to explain to anyone, without relying on Google:

When to use a class vs a function to create a component

The lifecycle of a class component

How to test a component

How to use a Jest spy to verify that a function is called correctly

What a Higher-Order Component (HOC) is and how to use it

How to optimize performance and control which components are rendered

Requirements

All files will be interpreted/compiled on Ubuntu 20.04 LTS using Node 20.x.x and npm 10.x.x

Allowed editors: vi, vim, emacs, Visual Studio Code

Each file must end with a newline

A README.md file at the root of the project folder is mandatory
